\sect{Summary from last lecture}\\
If we have a sequence: $_n\in \R$ if the lower limit of $\liminf a_n=\limsup a_n$ then the limit of $a_n$ exists and:
\[
  \liminf a_n=\limsup a_n=\lim a_n
\]
\newcommand{\nlim}{\lim_{n\rightarrow \infty}}
In $(L+\epsilon,\infty)$ There are finitley many $a_n$ \\
In $(-\infty,L-\epsilon,)$ There are finitley many $a_n$ \\
In $(L-\epsilon,L+\epsilon)$ There are almost all  $a_n$ which means there is some number N such that for all n>N $\a_n}\in (L-\epsilon,L+\epsilon)$\\
Thus the limit of the sequence exists
\\\\
Definition of Cauchy Sequence:
\[
  \forall \epsilon >0 \exists N : \forall m,n>N \quad |a_m-a_n|<\epsilon
\]
$a_n$ is cauchy if and only if $a_n$ converges to a number.\\
\[
  a_m-\epsilon \leq \liminf a_n \leq \limsup a_n\leq a_m+\epsilon
\]
since $a_m-\epsilon \leq \liminf a_n \leq \limsup a_n\leq a_m+\epsilon$ holds for all $\epsilon$ then $\liminf a_N=\limsup a_n$ which implies $a_n$ converges
